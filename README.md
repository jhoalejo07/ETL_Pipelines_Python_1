# ETL Pipeline in Python

## Overview
This project implements a modular ETL (Extract, Transform, Load) pipeline using Python.

## Architecture
- Extract: Fetches data from [source]
- Transform: Cleans and processes the data
- Load: Persists the data into [destination]

## Tech Stack
- Python
- Pandas
- Requests

## How to Run
```bash
python src/etl_pipeline/main.py
